## Parts of function:

1. Declaration: 
    return_type funcName();

2. Definition: 
    return_type funcName() { _ _ _};   [Giving parameters, Writing body]

3. Calling: 
    funcName();



## Types of functions:

1. No Return No Arugument: 
void mrg();

2. No Return With Arugument: 
void mrg(int x);

3. With Return With Argument:
int mrg(int x);

4. With Return No Argument:
int mrg();

# Escape Sequence: 
    An escape sequence is a special sequence of characters that starts with a backslash (\) and tells the compiler or interpreter to represent something non-printable or special in the output.
